개인정보처리방침

시행일: 2026년 7월 16일

BOOGIOS_STUDIO(이하 “운영자”)는 DayBlocks(이하 “앱”)를 이용하는 사용자의 개인정보를 소중히 다룹니다. 이 방침은 앱에서 어떤 정보를 처리하고, 왜 처리하며, 사용자가 어떻게 통제할 수 있는지 설명합니다.

1. 처리하는 정보

앱에서 직접 입력하거나 생성되는 정보

  • 사용자가 입력한 닉네임
  • DayBlocks에서 작성한 계획, 시간 블록, 메모, 완료 기록과 같은 앱 콘텐츠
  • Google Calendar 연결을 위해 앱에 저장되는 선택 캘린더 정보와 동기화 연결 정보

앱 이용 과정에서 처리되는 정보

  • 설치 식별자, 언어·지역 설정, 시간대, 운영체제, 앱 버전과 빌드 번호
  • 기능 사용 이벤트(예: 계획 생성·수정·삭제, 완료, 회고 저장, 설정 변경)와 이벤트 발생 시점
  • Google 계정 연결 상태와 사용자가 선택한 캘린더 정보. 연결된 계정 이메일은 연결 상태를 표시하기 위해 앱 화면에 표시될 수 있습니다.

2. Google Calendar 데이터의 사용

사용자가 Google 계정을 연결하고 권한을 허용한 경우에만 DayBlocks는 Google Calendar API를 사용합니다. 요청하는 권한은 다음과 같습니다.

  • calendar.events: 사용자가 선택한 캘린더의 일정을 읽고, DayBlocks와 Google Calendar 사이에서 일정 생성·수정·삭제를 동기화
  • calendar.calendarlist.readonly: 사용자가 동기화할 수 있는 캘린더 목록을 표시

Google Calendar 데이터는 이 기능을 제공하는 목적에만 사용합니다. 광고, 판매, 사용자 프로파일링, 또는 기능과 무관한 목적으로 사용하지 않습니다. 캘린더 목록과 일정 내용은 DayBlocks 서버나 Mixpanel로 전송하지 않으며, 사용자의 기기에서 Google Calendar API와 직접 동기화합니다.

Google 로그인과 액세스 토큰은 Google Sign-In SDK를 통해 처리합니다. 사용자는 언제든지 앱의 연결 해제 기능을 사용하거나 Google 계정의 서드 파티 앱 권한 페이지에서 DayBlocks의 접근 권한을 철회할 수 있습니다.

DayBlocks의 Google 사용자 데이터 사용은 Google API Services User Data Policy를 따릅니다.

3. 이용 목적과 처리 방식

  • 앱 콘텐츠를 저장하고 계획·실행·회고 기능을 제공하기 위해 처리합니다.
  • 사용자가 요청한 Google Calendar 연결과 양방향 일정 동기화를 제공하기 위해 처리합니다.
  • 앱 오류를 확인하고 기능 사용성을 개선하기 위해 최소한의 이용 이벤트와 앱 환경 정보를 분석합니다.
  • 서비스 문의에 답변하고, 서비스 운영에 필요한 안내를 제공하기 위해 처리합니다.

앱 프로필 등록을 위해 설치 식별자, 닉네임, 언어, 시간대, 플랫폼, 앱 버전과 빌드 번호가 Supabase를 통해 처리될 수 있습니다. 기능 이용 분석을 위해 Mixpanel이 사용자 번호, 닉네임, 앱 버전, 빌드 번호, 언어, 시간대와 이용 이벤트를 처리할 수 있습니다. 이 분석 데이터에는 Google Calendar 일정의 제목·시간·내용을 포함하지 않습니다.

4. 보관 및 삭제

계획, 메모와 동기화 연결 정보는 기본적으로 사용자의 기기에 저장됩니다. 앱을 삭제하거나 앱에서 연결을 해제하면 기기에 저장된 관련 정보가 삭제될 수 있습니다.

Supabase에 등록된 앱 프로필과 Mixpanel 분석 정보의 삭제를 요청하려면 아래 연락처로 앱 이름, 사용 중인 닉네임 또는 설치 정보를 함께 보내주세요. 확인 가능한 범위에서 요청을 처리합니다.

Google Calendar의 권한을 철회하면 DayBlocks는 이후 Google Calendar API에 접근할 수 없습니다. 이미 Google Calendar에 반영된 일정은 사용자가 Google Calendar에서 직접 확인·삭제해야 합니다.

5. 외부 서비스

  • Supabase: 앱 프로필 등록과 운영 지원
  • Mixpanel: 앱 이용 이벤트 분석
  • Google: Google 로그인 및 사용자가 요청한 Google Calendar 동기화

각 외부 서비스는 자체 개인정보처리방침과 보안 정책을 적용합니다. DayBlocks는 필요한 기능 제공 범위에서만 해당 서비스를 사용합니다.
